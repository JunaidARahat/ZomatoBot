# ZomatoBot


# Chainlit 

Chainlit is like Streamlit for LLM app development :)

To Know more about Chainlit, refer the official docs - https://docs.chainlit.io/overview



---


### CMD:

```bash
conda create -n zomato python=3.8 -y 
```

```bash
conda activate zomato
```

```bash
chainlit init
```

```bash
chainlit run app.py
```
